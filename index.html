PayFlip

PayFlip is a mobile application that allows users to authenticate with Google and send crypto payments across supported blockchains.

What we use Google Sign-In for

PayFlip uses Google Sign-In only to authenticate users. We access basic profile information such as name, email address and profile picture.

We do not read emails, access contacts or use any other Google services.

Platform

PayFlip is an Android application backed by a secure Rust-based API. All blockchain transactions are executed on-chain using user-controlled wallets.
